# 旅客行程智能推荐系统（前端部分）

## 介绍
软件杯大赛参赛[题目](http://www.cnsoftbei.com/bencandy.php?fid=155&aid=1702)  

前端部分由我完成，主要使用vue+vant组件库+高德地图api。

## 测试数据（中间站点也可以）

> 复兴门 -> 虹桥国际机场T2 （所有交通）  
> 万寿路 -> 菜市口 （地铁）    
> 复兴门 -> 正定国际机场 （地铁+高铁）
